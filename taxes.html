💰 세금 신고 (Non-Resident Alien)

F-1 학생도 미국에서 소득이 있으면(또는 없어도) 매년 세금 신고가 필요합니다. 흐름을 5단계로 정리.

⚠️ 본 페이지는 안내용. 본인 상황에 따라 다르므로 OIE / Sprintax 안내문을 반드시 함께 확인하세요. 거주자 신분(Resident vs Non-Resident)은 보통 5년차부터 바뀝니다.

1GLACIER 등록

📅 연말 ~ 연초 (TA/RA 시작 시)

학교에서 GLACIER 가입 메일이 옵니다. 정보 입력 후 W-4 / Form 8233 등을 출력해서 iStart에 업로드.

InputPassport · I-20 · I-94 · SSN
OutputW-4 · Form 8233 · Tax Summary Report
  • 이후 iStart → Tax Document Upload 제출

2소득 문서 자동 생성

📅 다음해 1월 말 ~ 2월
  • W-2 — OneUSG Payroll에서 발급 (급여 소득)
  • 1042-S — GLACIER에서 발급 (장학금/조세조약 소득)

3Sprintax 코드 받기

📅 2월 ~ 3월
  • iStart → Sprintax Access Request 제출
  • 학교에서 Sprintax access code 메일로 발송 (Federal 무료 / State는 본인 부담)

4Sprintax로 세금 신고서 작성

📅 2월 ~ 4월 (마감 4월 15일)
InputW-2 · 1042-S · Visa 정보
OutputForm 1040-NR · Form 8843 · GA State Return

5제출

📅 4월 15일까지
  • Federal (IRS) — Form 1040-NR + Form 8843 → 우편 제출 (Sprintax가 봉투 라벨까지 출력)
  • Georgia State — GA state return → e-file (Sprintax에서 바로)

주의사항

  • 소득이 없어도 Form 8843은 매년 제출해야 합니다 (F-1 학생 의무)
  • Sprintax 외 TurboTax 같은 일반 프로그램은 NR 학생에 부적합 (잘못 신고 가능)
  • 5년 이상 미국 거주 → Resident Alien으로 전환 → 그때부터 일반 1040 사용
  • 1042-S는 매년 늦게 발급되는 경우가 있어요 (2월 중순). 받기 전엔 Sprintax 진행 보류
📧 OIE 세금 안내 메일 + istart.gatech.edu 공지를 매년 1–2월에 꼭 확인하세요.
