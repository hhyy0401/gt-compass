수강신청 흐름

  1. Buzzport 로그인
  2. Registration (OSCAR) 클릭
  3. "Registration" 탭 → "Register for Classes"
  4. CRN 입력 또는 검색해서 추가
💡 수강신청 전: gt-scheduler.org에서 시간표 미리 짜보기 + critique에서 GPA 분포 확인 + 지도교수 승인.
  • 학비 납부 마감: 학기마다 다름. Bursar 페이지에서 확인
  • TA/RA로 funded면 자동 waive되는 부분 + 본인 부담 fee 구분 필요
  • 국제학생 의료보험비도 학비에 포함되어 청구 (waive 시 별도 처리)
